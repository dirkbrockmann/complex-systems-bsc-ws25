Complexity Explorables

The Complexity Explorables are a collection of interactive web-based illustrations and simulations that help to develop an intuition about complex systems concepts.

You can find them here: https://www.complexity-explorables.org.

The site went online in 2017 with the first explorable. Meanwhile the site hosts around 50 different explorables on various topics. Some illustrated famous models from complexity science, others display more recent models.

If you want an overview of all explorables, go to https://complexity-explorables.org/explorables/.

Open Source

Many explorables are available on github. I am currently working on making each Exporable’s source code available on github. If you are interested check the repositories here.
