# The Lab Course

<div class="mx-auto my-8 max-w-[70%] text-5xl rounded-xl p-6 flex flex-col items-center">

- Programming Complex Systems Simulations
- No programming skills are required
- Everyone needs a laptop
- Programming language: [**netlogo**](https://ccl.northwestern.edu/netlogo/) (free)

</div>


Netlogo is an agent based computer simulation framework that is easy to grasp and
very intuitive to use. It's famous of its little overhead for beginners and agent based
simulations. It's free and fun.
