The logistic map

June 2, 2025

This little adventure explores and explains the properties of a famous discrete time dynamical systems, known as the logistic map.

The logistic map looks like this:

xn+1=λxn(1−xn)withn=0,1,2,3...x_{n+1}=\lambda x_{n}(1-x_{n})\quad\text{with}\quad n=0,1,2,3...xn+1​=λxn​(1−xn​)withn=0,1,2,3...

Given a starting value 0≤x0≤10\leq x_{0}\leq10≤x0​≤1 and a positive and fixed parameter 0<λ<40 < \lambda <4 0<λ<4 the map produces a sequence of values

x0,x1,x2,...x_{0},x_{1},x_{2},...x0​,x1​,x2​,...

that you get by iterating it, e.g.

x1=λx0(1−x0)x_{1} =\lambda x_{0}(1-x_{0})x1​=λx0​(1−x0​) x2=λx1(1−x1)x_{2} =\lambda x_{1}(1-x_{1})x2​=λx1​(1−x1​) .........

and so forth.

Before we discuss some fascinating properties of this map, its richness, and set of unexpected behaviors, let’s first motivate it and connect it to applications and natural systems.

Motivation

It turns out that the logistic map can be motivated using a biological/ecological context.

Let’s assume we have a population of a species in which individuals replicate. Let’s say, for simplicity that in every generation n=0,1,2,...n=0,1,2,...n=0,1,2,... an individual has on average λ\lambdaλ offspring before it dies. This could be any positive number. It doesn’t have to be an integer because we think of this as an average or an expected number of offspring. So if say λ=1.5\lambda=1.5λ=1.5 then on average each individual has 1.51.51.5 offspring before it dies, for instance if in 50% of cases an individual has one offspring and in 50% of the cases 2 offspring, we get λ=1.5\lambda=1.5λ=1.5.

We call the parameter λ\lambdaλ the reproduction rate.

Now let’s say the population at time n=0n=0n=0 has UnU_{n}Un​ individuals. A generation later the population size will be

Un+1=λUnU_{n+1}=\lambda U_{n}Un+1​=λUn​

Therefore, if we start with an initial population U0U_{0}U0​ we get

Un=λnU0U_{n}=\lambda^{n}U_{0}Un​=λnU0​

which we can write as

Un=eln⁡λU0U_{n}=e^{\ln\lambda}U_{0}Un​=elnλU0​

So, depending on the value of λ\lambdaλ we can have two different scenarios.

  • If 0<λ<10 < \lambda < 10<λ<1 then ln⁡λ<0 \ln \lambda < 0lnλ<0 and thus Un+1<UnU _{n+1}<U _{n} Un+1​<Un​, the population dies out. This makes total sense if the typical number of offspring is less that one.
  • If λ>1\lambda>1λ>1 on the other we have ln⁡λ>0\ln\lambda>0lnλ>0 and thus Un+1>UnU_{n+1}>U_{n}Un+1​>Un​: the population grows indefinitely and exponentially.

So, if we only have replication and a reproduction rate λ>1\lambda>1λ>1 we expect exponential growth. Now, this can’t go on forever. Other factors will come into play if the population gets too big, e.g. limiting resources, competition, limited space, etc. One way to account for this is by saying that the reproduction rate λ\lambdaλ effectively depends in some way on the size of the population:

λ=λ(U)\lambda=\lambda(U)λ=λ(U)

We introduce a capacity CCC which we think of as a large population size that, if reached, reduces reproduction to zero or a very small number. Plus we would like to keep reproduction at some maximum value λ0\lambda_{0}λ0​ if U≪CU\ll CU≪C. What are the choices? Here’e one:

λ=λ0e−U/C\lambda=\lambda_{0}e^{-U/C}λ=λ0​e−U/C

In this case

λ≈λ0ifU≪C\lambda\approx\lambda_{0}\quad\text{if}\quad U\ll Cλ≈λ0​ifU≪C

and

λ≈0ifU≫C\lambda\approx0\quad\text{if}\quad U\gg Cλ≈0ifU≫C

An even simpler choice is a linear decrease

λ=λ0(1−U/C)\lambda=\lambda_{0}(1-U/C)λ=λ0​(1−U/C)

which is very similar but potentially easier to handle mathematically.

With this choice we get

Un+1=λ0(1−Un/C)UnU_{n+1}=\lambda_{0}(1-U_{n}/C)U_{n}Un+1​=λ0​(1−Un​/C)Un​

which we can iterated for a chosen set of parameters λ\lambdaλ and CCC and an initial population U0<CU_{0}<CU0​<C.

Now we can simplify this a bit by expressing the population as a fraction of the capacity CCC:

xn=Un/C.x_{n}=U_{n}/C.xn​=Un​/C.

Using the new quantity we get

xn+1=λ0(1−xn)xn.x_{n+1}=\lambda_{0}(1-x_{n})x_{n}.xn+1​=λ0​(1−xn​)xn​.

We can just drop the subscript 000 of λ0\lambda_{0}λ0​ and arrive at the logistic map

xn+1=λ(1−xn)xn.x_{n+1}=\lambda(1-x_{n})x_{n}.xn+1​=λ(1−xn​)xn​.

Let’s play with this equation a bit. In the panel below you can observe the sequence xnx_{n}xn​ for parameters x0x_{0}x0​ and λ\lambdaλ of your choice. If 1<λ<31 < \lambda < 31<λ<3 we get a sequence xnx_{n}xn​ that quickly approaches a fixed value.

If we increase the reproduction rate to λ>3.0\lambda>3.0λ>3.0 we observe a different behavior: The population doesn’t seem to equilibrate to a fixed value, but rather seems to oscillate in a period-2-cycle. Regardless of the initial condition x0x_{0}x0​ the system always approaches this cycle.

Let’s go to even higher values. For λ=3.8\lambda=3.8λ=3.8 we see even for very long sequences no apparent regular or periodic pattern of values emerges. The sequence of values xnx_{n}xn​ seems random and chaotic. In fact, the sequence never repeats and becomes aperiodic. More about this below.

One dimensional maps

In order to understand this behavior better, let’s back up a bit and think of a system like the logistic map in more general terms. The logistic map is an example of a one-dimensional discrete time map:

xn+1=f(xn)x_{n+1}=f(x_{n})xn+1​=f(xn​)

where in our case

f(x)=λx(1−x).f(x)=\lambda x(1-x).f(x)=λx(1−x).

The question that arises typically is how the system behaves for long times so when n→∞n\rightarrow\inftyn→∞.

Fixpoints

The first step we usually take is trying to identify special points, called fixed points or short fixpoints. These are values for xxx that remain invariant if we plug them into the iterated map, so:

xn+1=xnx_{n+1}=x_{n}xn+1​=xn​

and

xn=f(xn).x_{n}=f(x_{n}).xn​=f(xn​).

Iterated maps can have none, one or many such points and sometimes it’s difficult to compute them. In our case we can compute the fixpoints:

x=λx(1−x)x=\lambda x(1-x)x=λx(1−x)

which has solutions

x⋆=0x^{\star}=0x⋆=0

and

x⋆=1−1λ.x^{\star}=1-\frac{1}{\lambda}.x⋆=1−λ1​.

We see that the first one always exists and the second one exists if λ>1\lambda>1λ>1. So if we start with x0=x⋆x_{0}=x^{\star}x0​=x⋆ we will have

xn+1=x⋆x_{n+1}=x^{\star}xn+1​=x⋆

for all times. But what happens if we don’t start with x0=x⋆x_{0}=x^{\star}x0​=x⋆? Let’s try this with λ=5/2\lambda=5/2λ=5/2 and some x0x_{0}x0​ we see that after a few iterations the values of xnx_{n}xn​ are approaching the fixpoint

x⋆=1−1λ=3/5=0.6x^{\star}=1-\frac{1}{\lambda}=3/5=0.6x⋆=1−λ1​=3/5=0.6

This fixpoint seems to be attractive and we call it an attractor. Even if we start very close to the other fixpoint x⋆=0x^{\star}=0x⋆=0 say with x0=0.001x_{0}=0.001x0​=0.001 we see that the values xnx_{n}xn​ are repelled by it (we call that fixpoint a repeller) and attracted by the non-zero fixpoint.

Apart from doing the computations to determine the fixpoints we can find the fixpoints graphically. All we have to do is to draw the function y1=f(x)y_{1}=f(x)y1​=f(x) and the diagonal y2=xy_{2}=xy2​=x. When these two curves intersect we have

x=f(x).x=f(x).x=f(x).

For the logistic map we see that f(x)f(x)f(x) is an inverse parabola that meets the xxx-axis at zero and one, and intersects with the diagonal xxx only at x=0x=0x=0 if λ<1\lambda<1λ<1 and also at a non-trivial intermediate value if λ>1\lambda>1λ>1, consistent with the calculations we did.

Stability

There’s a way to determine the stability of a fixpoint, to check whether a fixpoint is an attractor or a repeller. Let’s assume that we have a fixpoint x⋆x^{\star}x⋆. And we start off with an initial value that’s a little bit off so that

x0=x⋆+δx0.x_{0}=x^{\star}+\delta x_{0}.x0​=x⋆+δx0​.

Now we compute

x1=f(x0)=f(x⋆+δx0).x_{1}=f(x_{0})=f(x^{\star}+\delta x_{0}).x1​=f(x0​)=f(x⋆+δx0​).

In the vicinity of the fixpoint, if δx0\delta x_{0}δx0​ is very small we can expand the function fff:

f(x⋆+δx0)≈f(x⋆)+f′ˆ(x⋆)δx0f(x^{\star}+\delta x_{0})\approx f(x^{\star}) + f\^{\prime}(x^{\star})\delta x_{0}f(x⋆+δx0​)≈f(x⋆)+f′ˆ(x⋆)δx0​

so that

x1≈x⋆ˆ+f′ˆ(x⋆)δx0x_{1} \approx x\^{\star}+f\^{\prime}(x^{\star})\delta x_{0}x1​≈x⋆ˆ+f′ˆ(x⋆)δx0​

So, after one iteration, what is the distance to the fixpoint?

δx1≈f′ˆ(x⋆ˆ)δx0.\delta x_{1}\approx f\^{\prime}(x\^{\star})\delta x_{0}.δx1​≈f′ˆ(x⋆ˆ)δx0​.

So we see that if

∣f′ˆ(x⋆ˆ)∣>1|f\^{\prime}(x\^{\star})|>1∣f′ˆ(x⋆ˆ)∣>1

this distance to the fixpoint becomes larger and larger whereas if

∣f′ˆ(x⋆ˆ)∣<1|f\^{\prime}(x\^{\star})|<1∣f′ˆ(x⋆ˆ)∣<1

it becomes smaller and smaller. So it seems that this a condition for the stability of a fixpoint.

Let’s try to apply this to the logistic map. Here we have

f(x)=λx(1−x)f(x)=\lambda x(1-x)f(x)=λx(1−x)

which implies that

f′ˆ(x)=λ−2λxf\^{\prime}(x)=\lambda-2\lambda xf′ˆ(x)=λ−2λx

so that at the fixpoint x⋆ˆ=1−1/λx\^{\star}=1-1/\lambdax⋆ˆ=1−1/λ we get

f′ˆ(x⋆ˆ)=λ−2(λ−1)=2−λf\^{\prime}(x\^{\star}) =\lambda-2\left(\lambda-1\right) =2-\lambdaf′ˆ(x⋆ˆ)=λ−2(λ−1)=2−λ

Therefore, if λ<3\lambda<3λ<3 the fixpoint is stable because ∣f′ˆ(x⋆ˆ)∣<1|f\^{\prime}(x\^{\star})|<1∣f′ˆ(x⋆ˆ)∣<1. But, if the reproduction rate increases beyond 333 the fixpoint becomes unstable. The system no longer has a stable, attractive fixpoint.

Looking at the sequence xnx_{n}xn​ say for λ=3.3\lambda=3.3λ=3.3 we see that the attractor is something else: a period-2-cycle of alternating values such that

xn+2=xn=xA⋆x_{n+2}=x_{n}=x_{A}^{\star}xn+2​=xn​=xA⋆​

and

xn+3=xn+1=xB⋆x_{n+3}=x_{n+1}=x_{B}^{\star}xn+3​=xn+1​=xB⋆​

We can think of both of these points that make that cycle as a fixpoint of a new map:

g=f∘fg=f\circ fg=f∘f

which in words means we apply the orginal map twice, or

g(x)=f(f(x))g(x) =f(f(x))g(x)=f(f(x)) =λf(x)(1−f(x)) =\lambda f(x)(1-f(x))=λf(x)(1−f(x)) =λ2x(1−x)(1−λx(1−x)) =\lambda^{2}x(1-x)(1-\lambda x(1-x))=λ2x(1−x)(1−λx(1−x))

This function is a more convoluted polynomial. If we want to compute the fixed points of g(x)g(x)g(x), we proceed the same way by solving

The road to chaos

So far we have observed that when λ>1\lambda>1λ>1 we have a single fixpoint at x⋆=1−1/λx^{\star}=1-1/\lambdax⋆=1−1/λ that loses stability at a critical value λ2=3\lambda_{2}=3λ2​=3 when a stable period-2-cyle emerges. If we increase λ\lambdaλ beyond λ3=3.449...\lambda_{3}=3.449...λ3​=3.449... we will see that the period-2-cycle also loses stability. It is replaced by a new, stable and attractive period-4-cycle. The points that make up this cycle are stable fix-points of the quadruple map

h=g∘g=f∘f∘f∘f.h=g\circ g=f\circ f\circ f\circ f.h=g∘g=f∘f∘f∘f.

Now it turns out that this period-4-cycle doesn’t remain stable for long but at λ4=3.54409...\lambda_{4}=3.54409...λ4​=3.54409... is replaced by a stable period-8-cycle, and so forth.

What’s important to note is that the critical points at which these period doubling events occur, or rather there separation becomes smaller and smaller and eventually at

λ∞=3.569946...\lambda_{\infty}=3.569946...λ∞​=3.569946...

we have a period of infinity. This means that although we have a “cycle” that is attractive, it’s period is infinite, so it never repeats, it is not longer really periodic. Rather is has become aperiodic and chaotic. This was very confusing to people that first looked at the simple logistic map. We can compute the ratio

δ=lim⁡k→∞λn−λn−1λn+1−λn=4.669\delta=\lim_{k\rightarrow\infty}\frac{\lambda_{n}-\lambda_{n-1}}{\lambda_{n+1}-\lambda_{n}}=4.669δ=k→∞lim​λn+1​−λn​λn​−λn−1​​=4.669

The limit is known as Feigenbaum’s constant. This is a universal constant that keeps reappearing in systems like the logistic map and is intimitaly connected to chaotic systems.

What’s even more puzzling is that the periodic solutions like the period-2-cycles, 4-cycles etc., although no longer stable in the chaotic regime, still exist as repellers. So for λ>λ∞\lambda>\lambda_{\infty}λ>λ∞​ we have a very dense and infinite set of periodic orbits in the system all of which are unstable. In fact, they have to fit in the system somehow, so actually for every point xxx we have an inifite set of periodic orbits arbitrarily close to it.

Bifurcation diagram

One way of illustrating the complexity of the situation is by what is known as a bifurcation diagram. Instead of making different cobwebbing plots for different choices of λ\lambdaλ and because we tend to be interested in the asymptotic behavior of the system we plot the attractors of the systems as a function of the parameter λ\lambdaλ. A plot like this is known as a bifurcation diagram, a very important tool in the study of dynamical systems. This is how it works:

  • first we fix λ\lambdaλ and choose an initial condition
  • we then iterate the map, say 10000 times for the transient part of the dynamics (which we then throw away)
  • then we iterate say 1000 more times and plot the array of values vs. λ\lambdaλ
  • we then increment λ\lambdaλ and repeat the process

we now plot the xxx-values that we computed that represent the attractor on the y−y-y−axis and the λ−\lambda-λ−value on the xxx-axis.

If for the chosen value of λ\lambdaλ we have say a period-8-cycle that is attractive, all the collected 1000 points will be one of the 8 points that make up the period-8-cycle. So we would get 8 dots in the plot. You can see what is happening in the panel below.

We clearly see the period doubling events as we increase λ\lambdaλ. We also see that at some point all the points are spread across a wide range and form what looks like a continuun. Furthermore we see that there are weird self-similar structures in the bifurcation diagram and also regimes of periodic behavior within a sea of chaos. You can investigate this by zooming by clicking on a region of interest.

We see another generic feature of chaotic systems: self-similarity. This is particularly striking in the bifurcation diagram where we see parts of the structure that resemble the whole structure only smaller. You can see this by zooming in (clicking on the region of interest).

Some properties of chaos

When the reproduction rate is larger than the critical parameter λ∞\lambda_{\infty}λ∞​ and we have a regime where orbits are aperiodic, we have reached the regime of deterministic chaos. This means that even though the orbits are deterministic, they seem irregular and unpredictable.

We can also observe a sensitive dependence on initial conditions. Let’s say we start with an initial value x0x_{0}x0​ and right next to it with an initial value x~_0=x_0+ε\tilde{x}\_{0}=x\_{0}+\varepsilonx~_0=x_0+ε where ε\varepsilonε is tiny, say 10−610^{-6}10−6. In the chaotic regime after a few steps the sequences xnx_{n}xn​ and x~n\tilde{x}_{n}x~n​ will become very different, another hallmark of unpredictability. You can check this out in the panel below.

Other systems

The logistic map with f(x)=λx(1−x)f(x)=\lambda x (1-x)f(x)=λx(1−x) is the simplest conceivable system that encorporates reproduction and self-regulation of a population. Because it is so simple we can expect that models that are structurally similar but more realistic generate similar behavior and that the funky effects observed in the logistic map are universal.

which isn’t limited to a parameter range for the reproduction rate λ<4\lambda<4λ<4 but is plausible for any non-negative value. The bifurcation diagram looks qualitatively similar to the one of the logistic map.
